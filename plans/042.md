## Key rules

- **Every** handler gets `exc_info=True` — no exceptions
- **Never** remove control flow: if the handler had `pass`, `continue`, `break`, or `return`, keep it after the log call
- For handlers that already have `print(f"... {e}")`, **replace** the print with the log call (don't add a second message)
- Use `%s` style formatting in log calls, not f-strings: `log.warning("context %s", var, exc_info=True)`

- ~~Remove the file~~ — loses documentation value for issue 013
- ~~Fix lint + skip all failing tests~~ — partial fix, many skips clutter test output
- **Fix lint/format + skip failing tests with clear reason** — cleanest approach that preserves test documentation while unblocking verification

**Why this over alternatives**:
- Absolute paths are acceptable because this is a local-only project (not shared/portable)
- Skipping (not deleting) premature tests preserves issue 013's acceptance test documentation
- When issue 013 is built, the skips are removed and replaced with real implementations
